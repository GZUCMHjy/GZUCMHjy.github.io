二分查找

  • 导论

    1. 概念：二分查找又称折半查找，通过不断改变两端边界的值，从而在一维数据中快速查找目标值。
    2. 使用前提：在具有单调(递增/递减)的一维数据(数组)。
    3. 思想：二分的过程就是不断微分的过程。

  • 边界问题

    细节：数组下标都是从零开始，故存进数组的数据也是从下标零开始。

  1. 搜索区间左闭右闭[l,r]—>寻找一个数

    l=0;
    r=数组长度-1;

    l r
    0 1 2 3 4 5 … n-1 
void BinarySearch(int arr[],int target){
    int l=0;//初始化搜索边界
    int r=n-1;//数组长度减一//下标为n-1未数组越界
        while(l<=r){//终止条件是(l==r+1)表示搜索区间是[r+1,r],此区间为开(不存在)
            int mid=l+(r-l)/2;//防止数据溢出
            if(a[mid]>target){
                r=mid-1;//[l,r]-->[l,mid-1]
            }
            else if(a[mid]<target){
                l=mid+1;//[l,r]-->[mid+1,r]
            }else{
                return l;//return r也可以
            }
        }
        return -1;//搜索区间为空,即找不到target则return-1
}
    1. 问题1：为什么while的循环条件是l <= r,而不是l < r?
      解: 因为前者的终止条件是l==r+1,搜索区间为[r+1,r]，此区间为空;后者终止条件是l==r,搜索区间为[r,r]，此区间不为空。当只有区间为空时，才可以退出while循环，表示未找到target值。
    2. 问题2：为什么称这搜索区间为左闭右闭?
      解：因为初始的左右下标边界都是可以取的，r=0;l=n-1(未越界)。
    3. 问题3：为什么r=mid-1,l=mid+1?
      解：因为已经检测到mid不是我们要找的target，故将mid除去。mid将初始化的搜索区间拆分为[l,mid-1],[mid+1,r];

  2. 搜索区间左闭右开[l,r)—>寻找左右侧边界

    void left_bound(int arr[],int target){
    int l=0;
    int r=n;
    while(l<r){//注意！！！
        int mid=l+(r-l)/2;
        if(a[mid]>target){
            r=mid;//注意！！！
        }else if(a[mid]<target){
            l=mid+1;
        }else{
            r=mid;//改变右侧边界r，向左靠拢
        }
    }return l;//return r也可以
}
void right_bound(int arr[],int target){
    int l=0;
    int r=n;
    while(l<r){//注意！！！
        int mid=l+(r-l)/2;
        if(a[mid]>target){
            r=mid;//注意！！！
        }else if(a[mid]<target){
            l=mid+1;
        }else{
            l=mid+1;//改变右侧边界l，向右靠拢
        }
    }return l-1;//return r-1也可以
}
      1. 问题1：为什么while的循环条件是l < r,而不是l <= r?
        解：初始化的搜索边界l=0；r=n;此时的右边界r=n是已经数组下标越界了，故将while循环的终止条件设置为l==r,即搜索区间为[r,r)，很明显此区间为空。
      2. 问题2.为什么l=mid+1，r=mid，而不是l=mid+1,r=mid-1?
        解：还是根据搜索区间置空的原理解释。检测mid不符合我们要找的target，所以将mid去掉，即mid将搜索区间分成两部分[l,mid),[mid+1,r)。
      3. 问题3：当target值为小于数组所有值或大于所有值或者介于端点值但不存在，那返回什么？
        解：问得好！当target小于任何数时，return 0；当target大于任何数时，return n；当target介于两端点值时，返回第一个大于target值的下标。
      4. 问题4：要想实现如果找不到target值就返回-1，怎么改？ 
            void left_bound(int arr[],int target){
    int l=0;
    int r=n;
    while(l<r){//注意！！！
        int mid=l+(r-l)/2;
        if(a[mid]>target){
            r=mid;//注意！！！
        }else if(a[mid]<target){
            l=mid+1;
        }else{
            r=mid;//改变右侧边界r，向左靠拢
        }
    }
    if(l==n)return -1;//当target大于任何数组的所有数时，返回-1，也就是说右侧边界r无法向左进行靠拢。//换成if(r==n)return -1也可以
return a[l]==target?l:-1;//换成return a[r]==target?r:-1也可以
}
void right_bound(int arr[],int target){
    int l=0;
    int r=n;
    while(l<r){//注意！！！
        int mid=l+(r-l)/2;
        if(a[mid]>target){
            r=mid;//注意！！！
        }else if(a[mid]<target){
            l=mid+1;
        }else{
            l=mid+1;//改变左侧边界l，向右靠拢
        }
    }
    if(l==0)return -1;//当target小于任何数组的所有数时，返回-1，也就是说左侧边界无法向右靠拢。//换成r==0也可以
return a[l-1]==target?(l-1):-1;//换成a[r-1]=target?(r-1):-1也可以
}

  • 总结

    1. 理解清楚while循环退出的条件(终止条件)
    2. 理解清楚搜索区间为空的作用
    3. 理解上面两条，你就明白如何确定二分的边界了！
