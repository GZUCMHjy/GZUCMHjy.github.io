JAVA WEB

  • 导论

    1. 概念：Java Web，是用Java技术来解决相关web互联网领域的技术栈。web包括：web服务端和web客户端两部分。(B/S或者C/S)
    2. WEB分类：动态WEB资源(指web页面中供人们浏览的数据是由程序产生的，不同时间点访问web页面看到的内容各不相同。);静态WEB资源(指web页面中供人们浏览的数据始终是不变。)。
    3. 开发技术: HTML、CSS(静态web资源开发)；JavaScript、JSP/Servlet、ASP、PHP(动态web资源开发)。

  • 前端开发三件套

    1. HTML

    1. 作用

      用于处理文字，画面排版，图片显示等

    2. HTML文件固定书写格式

    <html><!--html页面的开始-->
   <head><!--头标题的开始-->
      <title>标签</title>
   </head><!--头标题的结束-->
   <body><!--主题内容的开始-->

      .....<!--页面主题内容-->

   </body><!--主题内容的结束-->
</html><!--html页面的结束-->

    1. HTML标签介绍

      1. 标签格式

        <标签名>封装的数据</标签名>

      2. 标签名对大小写不敏感
      3. 标签属性
        1. 基本属性 如：< body bgcolor=”red”>< /body>
        2. 事件属性 如：< button onclick=”alert(‘hello,world’)”>< /button>
      4. 标签种类
        1. 单标签：<标签名/> 如：< br/>换行 < hr/>分隔符
        2. 双标签：<标签名>封装数据</标签名> 如：< body >< /body >
      5. 标签语法
        1. 标签不能交叉嵌套
        2. 标签必须完整(双标签必须成对出现,单标签必须完整(< br/>))
        3. 属性赋值必须要带双引号 “”

    2. 基本标签

      1. font标签 
        <font color="red" face="宋体" size=5>我是字体标签</font>
<!--
   >> 1. font标签(基本属性)
       1. size (1~7)
       2. face (字体样式)
       3. color
   -->
      2. 标题标签(< h1>标题内容< /h>)
        1. h1~h6
        2. align=”left”/“right”/“center”(对齐属性)
      3. 超链接标签() 
        <html>
   <head>
      <title>百度</title>
      <body>
         <a href="https://www.baidu.com">百度</a>
         <!--
             基本属性
               1. href 存放地址 如:https://www.baidu.com
               2. target: _shelf / _blank
               3. 列表标签(无序列表/有序列表)
         -->
      </body>
   </head>
</html>
      4. 列表标签 
        <html>
   <head>
      <title>百度</title>
      <body>
         <ul><!--无序列表的标签 u表示乱序-->
            <li>小文</li><!--li表示list item--><!--li表示列表项-->
            <li>小红</li>
            <li>小君</li>
            <li>小名</li>
         </ul>
         <ol><!--有序列表的标签--><!--o表示order-->
            <li>小文</li><!--li表示list item-->
            <li>小红</li>
            <li>小君</li>
            <li>小名</li>
         </ol>
      </body>
   </head>
</html>
      5. 图片标签 (单标签) 
        <html>
   <head>
      <title>百度</title>
      <body>
         <!--
         基本属性
            1.src=""设置图片路径
            2.width
            3.height
            4.boder(边框)
         web接受路径有两个
            1.相对路径：
               . 表示当前文件所在的目录
               .. 表示当前文件夹上一级的目录
               文件名 表示当前文件所在目录的文件 .文件名/ (可以省略)
            2.绝对路径
               正确格式:https://ip:port/工程名/资源路径
         -->
         <img src="../img/landscape_sky2.jpg" width="100" height="200" boder="1"/>
      </body>
   </head>
</html>
      6. 表格标签 
        <html>
   <head>
      <title>标题</title>
      </head>
      <body>
         <!--
            table 是表格标签
               1.align 对齐属性
               2.width
               3.height
               4.boder  
               5.ceilspacing 单元格间距
            tr 表示行标签
            th 表示头标签 默认居中
               1.colspan 跨行
               2.rowspan 跨列
            td 表示单元格标签
               1.colspan 跨行
               2.rowspan 跨列
         -->
         <table border="1" width="500" height="500" ceilspacing="0">
            <tr>
               <th colspan="2">1.1</th>
               
               <th>1.3</th>
               <th>1.4</th>
            </tr>
            <tr>
               <th>2.1</th>
               <th>2.2</th>
               <th>2.3</th>
               <th>2.4</th>
            </tr>
            <tr>
               <th>3.1</th>
               <th>3.2</th>
               <th>3.3</th>
               <th>3.4</th>
            </tr>
         </table>
      </body>
</html>
      7. iframe标签 
        <html>
      <head>
         <title>百度</title>
         <body>
            <!--iframe独立生成一个页面-->
            <!--iframe和a标签的应用-->
            <!--
               1.给iframe命名，给静态资源链接(src)
               2.给超文本(href)标签a提供跳转目标target，target与name同名
               -->
            <iframe src="../img/landscape.jmg" width="500" height="500" name="..."></iframe>
               <ul>
                  <li><a href="..." target="...">....</a></li>
                  <li><a href="..." target="...">....</a></li>
                  <li><a href="..." target="...">....</a></li>
                  <li><a href="..." target="...">....</a></li>
               </ul>
         </body>
      </head>
   </html>
